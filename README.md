## 🧠 これは何？

「じゃんけん」（グー・チョキ・パー）をベースにした、**CLI向けの対戦型ゲーム**です。

- 🪨📄✂️ 直感的なターン制バトル  
- ASCIIスタイルのUI & アニメーション  
- Tetris[効果音](https://www.zophar.net/music/gameboy-gbs/tetris-dx)
- 日本語対応 & 美しい端末体験
